# GPU Catalog (Legacy version)

## About

GPU Catalog is an HTML page to display detailed information for various graphics cards from all 3 major manufacturers.

The legacy version of the app, using only HTML + CSS, without any APIs to fetch data.

## Usage

Open the `index.html` file on your favorite browser.

OR

Use a Python HTTP local server
```bash
python -m http.server
```

- Access the website on http://localhost:8000
